Welcome to dev version of vocloud server – distributed parallel system for machine learning of astronomical data.

The early stages of development of this service in years 2013–2015 was supported by grant 13-08195S of Czech Science Foundation.

Its current extended version has been funded by grant COST LD-15113 of Czech Ministry of Education Youth and Sport.

Source code is available on github.
